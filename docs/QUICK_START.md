# Quick Start Guide

Get up and running with SEO AI Meta Generator in 5 minutes!

## Step 1: Install & Activate

1. Upload plugin to `/wp-content/plugins/seo-ai-meta-generator/`
2. Activate in WordPress Admin > Plugins
3. You're ready to go!

## Step 2: Generate Your First Meta Tag

1. **Edit any post** in WordPress
2. Find the **"SEO AI Meta Generator"** meta box
3. Click **"Generate Meta"**
4. Review the generated title and description
5. Save your post

That's it! Your meta tags are now live.

## Step 3: Configure Settings (Optional)

1. Go to **Posts > SEO AI Meta > Settings**
2. (Optional) Enter OpenAI API key
   - Leave empty to use Render credentials
   - Or add your own key for more control
3. Adjust character limits if needed
4. Save changes

## Step 4: Bulk Generate (Optional)

To process multiple posts at once:

1. Go to **Posts > Bulk Generate Meta**
2. Select posts without meta tags
3. Click **"Generate Meta for Selected Posts"**
4. Watch the progress bar
5. Done!
